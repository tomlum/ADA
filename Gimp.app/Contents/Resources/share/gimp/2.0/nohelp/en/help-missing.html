Eeek!  GIMP help files are not installed.

Sorry, but help files are not installed on this system. To get access to the help files you can either

Download help files

and install them locally on your Mac computer. This may be especially useful if you run the GIMP without your Mac being connected to the internet. Please download the help package mentioned on the GIMP on OS X website and run the installer app to install it on your Mac. Or

Use the online manual

You may be able to find the information you are looking for in the online version of the help at the GIMP docs website. And you can use the menu entry "Help -> GIMP online" too.

And finally

please notice, that help is currently work in progress.

Feel free to join us and fill the gap by writing documentation for GIMP. For more information, subscribe to our Mailing list. Generally, it's a good idea to check the GIMP project page.
